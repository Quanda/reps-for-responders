## Usage
### Running the app in Development
  Start the development server
  ```
  yarn dev
  ```
  Open the app in your browser
  ```
  http://localhost:8000/
  ```

***********************************************

### Build the app for Production
  Build the app
  ```
  yarn build
  ```

***********************************************

### Clear the cache and public directories
  Useful to clear stale data
  ```
  yarn clean
  ```
